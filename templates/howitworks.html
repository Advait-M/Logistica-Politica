How Logistica Politica Works?


The Power of Social Data

Our web service isn't one that satisfies a need but rather it's one that satisfies the burning want: the curiosity to find out where you, your friends and your political leaders stand on the political scale. Furthermore, the ability to see the emotions attached to that political interest further satisfies curiosity. With almost a year of anticipation for the 2016 US Elections, that interest only increases.

The Twitter API

Using a twitter API, we can collect data from tweets. When a user inputs a valid twitter account into our web service, the API collects string information from that account's tweets. Our next API (mentioned below) acts upon this information.

The Indico API

Indico is a "machine-learning" API that makes decisions on an input to produce an output that shows a trend. The string from the twitter API is inputted into the Indico API where it tells a user his/her political standing on the spectrum and his/her sentiment towards that standing.
